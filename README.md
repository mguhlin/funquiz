# funquiz

🎓 **Fun Personality Quizzes for Educators**

This project is a fun, interactive quiz website that helps educators discover which iconic character best matches their teaching style. Each quiz is themed around a beloved franchise:

- 🐻 **Winnie the Pooh**
- 🔵 **The Smurfs**
- 🍎 **Snow White and the Seven Dwarfs**
- 🎯 **Looney Tunes**

---

## 🚀 How to Use

1. Clone or download this repository.
2. Open any of the following HTML files in your browser:
   - `index.html` (Winnie the Pooh)
   - `smurfs.html`
   - `dwarfs.html`
   - `looney.html`
3. Each quiz contains 5 multiple-choice questions.
4. At the end, you'll receive your character match instantly!
